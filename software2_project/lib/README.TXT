How to compile?
To compile the project, you need to have a virtual machine emulator like (Android Studio) to run the project on, you also need to install the Flutter and Dart extensions in the IDE (Visual Studio Code is preferred) you are using. 
Make sure to connect the emulator to the IDE.

Steps to run the app:
1.	Emulator to run the project on (Android Studio is preferred) 
2.	installing Flutter extension on your IDE (Visual Studio Code IDE is preferred)
3.	installing Dart extension on your IDE.
4.  hit the 'run without debugging' button.
5.  the app should run and appear on your Emulator.
